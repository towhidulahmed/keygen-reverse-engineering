# KeygenMe3 Reverse Engineering - Summary

## What This Is

Reverse engineering project for the keygenme3 binary. Created a working keygen.

## Quick Info

- Binary: keygenme3 (ELF 64-bit, stripped)
- Tool: Ghidra
- Result: Working keygen in cracker.c
- Test case: pr0cracker -> 1f68-190d9

## The Algorithm

For each character in email:
    local_44 += character * 8
    local_40 += (character * character) - 0xbc

serial = sprintf("%04x-%04x", local_44, local_40)
